# SynPortScan
### [⚠️] Experimental project

### Parameters
```
Description:
  SynPortScan args.

Usage:
  SynPortScan [options]

Options:
  --ip <ip>            Target IP
  --threads <threads>  Threads (optional)
  --version            Show version information
  -?, -h, --help       Show help and usage information
```
### Sample
```
dotnet run --project src -- --ip scanme.nmap.org
```
